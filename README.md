## RD-Pastebin

My own implementation of pastebin, note-dropper or shared clipboard. Intended to be hosted at a home-server

## Features

1. Adding pastes
2. Deleting pastes
3. Editing pastes

## Running

1. Install dependencies
   
   ```bash
   python3 install -r requirements.txt
   ```

2. Run the app in the background 
   
   ```bash
   nohup fastapi run &
   ```
